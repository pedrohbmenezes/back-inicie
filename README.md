
## Description
Application(nest.js) created for the challenge of the company "start",
the same consumes a user registration api and creation of posts and comments.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Manual
All requests are documented and have a postman file collection
